3️⃣ Arquitectura de Capas

🌐 Capa Central (Servidor)

  • MQTTFedAvgSwell / MQTTFedAvgSweet
  • Extiende: FedAvg (Flower)
  • Agrega partiales de bridges
  • Publica global model por MQTT
  • Protocolo: Flower gRPC (entrada), MQTT (salida)

🌉 Capa Fog Bridge

  • FogClientSwell / FogClientSweet
  • Implementa: NumPyClient (Flower)
  • Escucha fl/partial (MQTT)
  • Forwarding a servidor (gRPC)
  • 1 instancia por región

🔗 Capa Fog Broker

  • fog.py / sweet_fog.py
  • Módulos MQTT puros (no clases)
  • Bufferiza K clientes
  • Agregación con NumPy (FedAvg)
  • Publica partiales a bridges

📱 Capa Edge (Clientes Locales)

  • SwellFLClientMQTT / SweetFLClientMQTT
  • Hereda: BaseMQTTComponent
  • Entrena localmente
  • Publica en fl/updates
  • Escucha fl/global_model

📌 Relaciones de Herencia

BaseMQTTComponent: Clase base abstracta que proporciona comunicación MQTT. Define el método hook on_message() que es implementado por subclases.

FogClientSwell/Sweet: Hereda de BaseMQTTComponent (MQTT) E implementa NumPyClient (Flower). Esto permite escuchar MQTT y al mismo tiempo ser un cliente Flower.

MQTTFedAvgSwell/Sweet: Extiende FedAvg de Flower. Override de métodos para agregar publicación MQTT del modelo global.

🔄 Flujo de Comunicación

Edge → Broker: MQTT topic fl/updates
Clientes publican pesos locales + número de muestras
Broker → Bridge: MQTT topic fl/partial
Broker publica pesos agregados por región (NumPy FedAvg)
Bridge → Server: Flower gRPC
Bridge cliente implementa NumPyClient.fit() → comunica con servidor Flower
Server → Edge: MQTT topic fl/global_model
Servidor publica modelo global para siguiente ronda

⚙️ Métodos Clave por Capa

Capa Clase Método Principal Descripción
Edge SwellFLClientMQTT train_local() Entrena modelo localmente, retorna pesos
Edge SwellFLClientMQTT publish_update() Publica update en MQTT fl/updates
Broker fog.py on_update() Recibe update, bufferiza, agrega cuando hay K
Broker fog.py weighted_average() Promedio ponderado con NumPy (FedAvg)
Bridge FogClientSwell fit(parameters, config) Forwarding del partial al servidor
Bridge FogClientSwell on_message() Recibe partial del broker en MQTT
Central MQTTFedAvgSwell aggregate_fit() Agrega partiales, crea modelo global
Central MQTTFedAvgSwell publish_global_model() Publica modelo global en MQTT
